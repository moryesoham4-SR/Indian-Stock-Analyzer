# Indian Stock Analyzer

## Live Demo

https://indian-stock-analyzer-ztt72jzvbmtgrasvvufzkg.streamlit.app/

## Overview

Indian Stock Analyzer is a web-based stock market analytics platform built using Python, Streamlit, Machine Learning, and MySQL. It provides real-time stock analysis, technical indicators, AI-powered forecasting, portfolio tracking, and market insights for Indian stock market investors.

## Features

* Real-time Stock Data
* Technical Analysis (RSI, MACD, Bollinger Bands)
* AI Stock Price Prediction
* Fundamental Analysis
* Portfolio Tracker
* Market Dashboard
* Stock Comparison
* Watchlist Management
* Sector Analysis
* Stock Screener
* SIP Calculator
* Earnings Calendar
* Latest Stock News
* PDF Report Generation
* Cloud MySQL Integration

## Technologies Used

* Python
* Streamlit
* Pandas
* NumPy
* Plotly
* Scikit-Learn
* MySQL
* Yahoo Finance API (yfinance)

## Machine Learning

The application uses Linear Regression models to analyze historical stock data and forecast future stock price trends.

## Database

MySQL is used for storing and retrieving historical stock records, enabling efficient data management and analysis.

## Objective

To provide investors with a single platform for stock analysis, investment tracking, technical indicators, AI-based forecasting, and market research.

## Future Enhancements

## Conclusion

Indian Stock Analyzer combines financial analytics, machine learning, and interactive dashboards to deliver a complete stock market analysis solution for investors and traders.
