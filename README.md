# Indian Stock Analyzer

## Overview

Indian Stock Analyzer is a comprehensive stock market analytics platform built using Python, Streamlit, Machine Learning, and MySQL. The application provides real-time stock analysis, technical indicators, AI-powered forecasting, portfolio management, and market insights for Indian stock market investors.

## Live Demo

https://indian-stock-analyzer-ztt72jzvbmtgrasvvufzkg.streamlit.app/

## Features

* Real-Time Stock Market Data
* Technical Analysis (RSI, MACD, Bollinger Bands)
* AI Stock Price Prediction
* Fundamental Analysis
* Portfolio Tracker
* Stock Comparison
* Watchlist Management
* Market Dashboard
* Stock Screener
* Sector Analysis
* SIP Calculator
* Earnings Calendar
* Latest Stock News
* PDF Report Generation
* Cloud MySQL Database Integration

## Tech Stack

* Python
* Streamlit
* Pandas
* NumPy
* Plotly
* Scikit-Learn
* MySQL
* Yahoo Finance API

## Machine Learning

The project uses Linear Regression models to analyze historical stock data and forecast future stock price trends.

## Database

MySQL is used to store and manage historical stock records, enabling efficient retrieval and analysis.

## Installation

```bash
git clone https://github.com/moryesoham4-SR/Indian-Stock-Analyzer.git

cd Indian-Stock-Analyzer

pip install -r requirements.txt

streamlit run app.py
```

## Project Goal

To provide investors and traders with a single platform for stock analysis, technical indicators, AI-based forecasting, portfolio tracking, and market research.
